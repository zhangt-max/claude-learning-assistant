## 项目简介

## 使用方法

### 安装配置

1. **环境要求**
   - Node.js 18.0+
   - npm 或 yarn

2. **安装步骤**
   ```bash
   # 克隆项目
   git clone <repository-url>
   cd claude-learning-assistant

   # 安装依赖
   npm install

   # 配置环境变量
   cp .env.example .env
   # 编辑 .env 文件，填入 API Key
   ```

3. **API Key 获取**
   - 访问 [Anthropic Console](https://console.anthropic.com/)
   - 创建 API Key 并配置到 `.env` 文件

### 功能使用

1. **启动应用**
   ```bash
   npm start
   ```

2. **主菜单功能**
   - 1 📖 代码解释 - 粘贴代码获取详细解释
   - 2 📚 概念讲解 - 学习 JavaScript 核心概念
   - 3 💻 代码生成 - 根据需求生成代码
   - 4 🎓 互动问答 - 苏格拉底式学习引导
   - 5 📊 学习统计 - 查看使用历史和成本
   - 6 ❌ 退出程序

3. **通用命令**
   - `/help` - 显示帮助信息
   - `/clear` - 清除对话历史
   - `/summary` - 查看会话摘要
   - `/export` - 导出对话记录
   - `/exit` - 返回主菜单

### 成本控制

- **每日预算**：默认 $0.50，可在 `.env` 中调整
- **实时监控**：每次调用显示 Token 使用量
- **警告机制**：达到预算 80% 时提醒
- **模型选择**：支持不同成本的模型切换
